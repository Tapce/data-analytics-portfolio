## Why this folder exists

GitHub doesn't render `.twbx` files inline, and Tableau Public dashboards are interactive — they belong on Tableau Public, not in the repo. This folder holds:

- Dashboard screenshots (PNG)
- Source data (where licensable)
- Brief write-ups of the question, approach, and findings for each dashboard
